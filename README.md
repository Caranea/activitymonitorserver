# Web usage tracker backend

Backend for chrome extension to track internet browsing habits

Extension repo: https://github.com/Caranea/webUsageTracker

### Version: 1.0.0

### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000

# Unit tests:
$ npm run test:unit
```
